# MyCalculator

## О проекте

MyCalculator — это современный веб-калькулятор с эффектным неоновым дизайном.  
Проект создан для демонстрации навыков frontend-разработки, включая HTML, CSS (с плавными неоновыми анимациями) и JavaScript.  

## Основные возможности

- Выполнение базовых арифметических операций: сложение, вычитание, умножение, деление.
- Современная тёмная и светлая тема с плавным переключением.
- Адаптивный дизайн.
- Неоновый стиль с подсветкой кнопок и эффектами свечения.
- Удобный и понятный интерфейс с крупным дисплеем результата.

## Технологии

- HTML5 — структура страницы и элементов калькулятора.
- CSS3 — стилизация, неоновые тени, анимации и адаптивность.
- JavaScript — обработка логики калькулятора и переключение тем.

## Установка и запуск

1. Клонируйте репозиторий:  
   `git clone https://github.com/LesanePC/MyCalculator.git`
2. Откройте файл `index.html` в браузере или запустите проект через локальный сервер.
3. Для разработки используйте любой редактор кода, например VSCode.

## Как использовать

- Нажмите на кнопки с цифрами и арифметическими операциями.
- Для результата нажмите "=".
- Переключение темы через кнопку "Сменить тему" под калькулятором.
- Обновите страницу для сброса.

## Планируемые улучшения

- Поддержка истории вычислений.
- Поддержка более сложных вычислений (например, скобки).
- Добавление звуковых эффектов при нажатии кнопок.
